# Webpage to Markdown

Chrome extension that converts any webpage to clean Markdown in one click.

## Features

- **Popup** — convert, preview, copy, or download the current page as Markdown
- **Context menu** — right-click → *Copy page as Markdown* copies directly to clipboard
- Smart content extraction: strips nav, ads, scripts, and UI chrome
- Powered by [Turndown](https://github.com/mixmark-io/turndown)

## Install (dev)

```
chrome://extensions  →  Developer mode  →  Load unpacked  →  select this folder
```
